Este archivo reúne varios ejercicios resueltos en Python para practicar la resolución de problemas mediante código y consolidar los fundamentos del lenguaje.
Incluye ejemplos con:
- Estructuras de datos básicas: listas, diccionarios, tuplas y sets.
- Condicionales y estructuras de control de flujo.
- Bucles e iteraciones.
- Funciones reutilizables para automatizar tareas concretas.
