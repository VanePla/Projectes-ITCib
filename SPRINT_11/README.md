Este proyecto reúne diferentes ejercicios centrados en la visualización de datos con Python, integrando herramientas como Pandas, Matplotlib y Seaborn, además de su conexión con MySQL y su presentación en Power BI.

Incluye ejemplos para:
* Comprender los principales métodos de visualización de datos.
* Crear gráficos sencillos e interactivos con las librerías más utilizadas en Python.
* Escoger las herramientas de visualización más adecuadas según la situación.
* Integrar soluciones que combinan Python con MySQL y Power BI para generar informes.
