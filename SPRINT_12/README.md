Este proyecto reúne varios ejercicios prácticos para aprender a consumir APIs REST con Python desde Jupyter Notebook, trabajar con peticiones HTTP y transformar las respuestas en datos analizables con pandas.

Incluye ejemplos para:
* Identificar y utilizar distintas APIs públicas a partir de su documentación y endpoints.
* Practicar métodos HTTP (GET, POST, DELETE, etc.) e interpretar codigos de estado del servidor.
* Convertir respuestas JSON en DataFrames de pandas y guardarlas para análisis posterior.
