Este archivo reúne varios ejercicios resueltos con la librería Pandas para practicar la limpieza, transformación y análisis de datos dentro del ecosistema de Python.

Incluye ejemplos para:
* Crear, importar y modificar DataFrames.
* Ordenar y transformar datos para obtener resúmenes útiles.
* Aplicar analíticas específicas sobre conjuntos de datos.
* Automatizar procesos y abordar problemas de datos más complejos
